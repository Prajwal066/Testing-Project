# QA Project — Web Application Testing & Automation (Sauce Demo)

## Overview
End-to-end QA testing and automation of an e-commerce web application, covering functional validation, defect identification, test execution reporting, UI automation (Python, pytest, Playwright), and API testing.

---

## Application Under Test
- Web Application: Sauce Demo  
- API: ReqRes  

---

## Tech Stack

**Automation:** Python, pytest, Playwright  
**API Testing:** requests  
**Manual Testing:** Test Case Design, Execution, Bug Reporting  
**Tools:** Git, CSV, Chrome  

---

## Test Scope

The following functional areas were tested:

- User authentication (login)
- Product inventory
- Cart functionality
- Checkout process
- API endpoints (GET, POST, negative scenarios)
